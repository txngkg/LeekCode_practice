# 1456.定长子串中元音最大数目

## 题目描述

给你字符串 `s` 和整数 `k` 。

请返回字符串 `s` 中长度为 `k` 的单个子字符串中可能包含的最大元音字母数。

英文中的 **元音字母** 为（`a`, `e`, `i`, `o`, `u`）。

 

**示例 1：**

```
输入：s = "abciiidef", k = 3
输出：3
解释：子字符串 "iii" 包含 3 个元音字母。
```

**示例 2：**

```
输入：s = "aeiou", k = 2
输出：2
解释：任意长度为 2 的子字符串都包含 2 个元音字母。
```

**示例 3：**

```
输入：s = "leetcode", k = 3
输出：2
解释："lee"、"eet" 和 "ode" 都包含 2 个元音字母。
```

**示例 4：**

```
输入：s = "rhythms", k = 4
输出：0
解释：字符串 s 中不含任何元音字母。
```

**示例 5：**

```
输入：s = "tryhard", k = 4
输出：1
```

 

**提示：**

- `1 <= s.length <= 10^5`
- `s` 由小写英文字母组成
- `1 <= k <= s.length`



## 解答

```c
//定长滑动窗口
//1.入
//2.更新
//3.出
int max(int a,int b){
  if(a>b) return a;
  else return b;
}
int maxVowels(char* s, int k) {//为什么不传数组的长度？字符串是以'\0'做结束标志的字符数组，函数不需要额外传递数字长度。s是字符串数组即是char s[]={"abc"}这样定义的，所以会有'\0'结束符。char a[3]={'a','b','c'}普通字符串，没有结束符。
  //入
  int i,count=0,ant=0,left;
  for(i=0;s[i];i++){//为什么可以用s[i]做终止条件？因为自带'\0'结束符    if(s[i]=='a'||s[i]=='e'||s[i]=='i'||s[i]=='o'||s[i]=='u')
  {
    count++;
    }
    left=i-k+1;
  if(left<0){
      continue;
    }
//更新
  ant=max(ant,count);
  if(ant==k){
     break;
   }
   //出
  char out=s[left];    if(out=='a'||out=='e'||out=='i'||out=='o'||out=='u'){
    count--;  
}
  }
  return ant;
}
```
